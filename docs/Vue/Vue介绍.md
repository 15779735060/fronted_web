### 概述
&emsp;&emsp; [Vue](https://v2.cn.vuejs.org/) 是一套用于构建用户界面的渐进式框架。使用MVVM体系结构，是目前生产环境中使用最广泛的JavaScript框架之一，几乎不需要手动优化，并且完全有能力处理大规模的应用。下面会对vue的核心模块，结合源码做出详解，以及对周边生态插件库使用方式。
